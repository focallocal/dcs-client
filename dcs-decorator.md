# dcs-decorator.js

## How to use it

Add the following lines to the `<head>` section of your web pages:

```html
<link
  rel="stylesheet"
  type="text/css"
  href="https://sylque.github.io/dcs-client/css/dcs-decorator.css"
/>
<script
  src="https://sylque.github.io/dcs-client/dist/dcs-decorator.js"
  async
></script>
```
